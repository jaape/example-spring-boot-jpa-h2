# Example project to test JPA and H2
Example project to test JPA (+Hibernate) and H2.

Data migrations are done by running CommandLineRunner (DataMigration.java) every time the app starts

Commit https://github.com/jaape/example-spring-boot-jpa-h2/pull/5/commits/558638c0c8782d502f062affe736664205f2f621 introduces a feature to use flyway migrations instead.
